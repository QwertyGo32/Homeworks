Теги HTML
Тег Описание
<!DOCTYPE> предназначен для указания типа текущего документа — DTD
(document type definition, описание типа документа)
<a> предназначен для создания ссылок
<b> устанавливает жирное начертание шрифта
<br> устанавливает перевод строки в том месте, где этот тег находится
<button> создает на веб-странице кнопки
<body> предназначен для хранения содержания веб-страницы (контента),
отображаемого в окне браузера
<div> является блочным элементом и предназначен для выделения
фрагмента документа с целью изменения вида содержимого
<h1>...<h6> шесть заголовков разного уровня
